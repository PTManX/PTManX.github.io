CEMU--塞尔达传说-简要教程
2020-02-24|CEMU模拟器
阅读量:|评论数:

配置参考：

本人电脑配置为：

  • cpu：I5-8400
  • 显卡：1060
  • 内存：16g

塞尔达1080p分辨率下稳定55帧左右，4k分辨率稳定30帧左右

博客仅提供教程，游戏资源请进群或自己从其它网站自取

初步设置

1.下载解压完成后，先不要运行，确保复合包（Cemu&BotW所在路径尽量，尽量不包含中文路径，建议存在固态硬盘中。

2.打开复合包（Cemu&BotW中的cemu文件夹，右键点击 Cemu. exe发送到桌面快捷方式，以便日后使用；同样地，右键点击 Cemu. exe，选择属性，按照下图设置（此为 Windows 1909版本的设置界面）

3.运行一次 Cemu. exe然后关闭，若不能运行，尝试安装

Cemu& BotW Resources\vc _redist x64.exe（resources文件夹内）

4.NVIDIA 显卡用户请打开 NVIDIA 控制面板，选择左侧的管理 3D 设置，然后 选择程序设置，点击添加，找到 Cemu 的图标如下 ，选择添加选定程序

若无法找到 Cemu 图标，则点击浏览，找到复合包中的 Cemu.exe 并确定 按照下图设置（三重缓冲和线程优化请务必确保开启）

5.暂未测试 AMD 显卡，AMD 显卡用户可参考下图

6. 将系统的电源计划调整至 高性能

7.确定系统的物理内存大小，建议](https://jingyan.baidu.com/article/2fb0ba4041d14c00f3ec5f4e.html)[ ](https://jingyan.baidu.com/article/2fb0ba4041d14c00f3ec5f4e.html)16GB 及以上，若物理内存不足 16GB，建 议 设置虚拟内存 ，确保物理内存+虚拟内存 ≧ 16GB

CEMU设置

1.打开 Cemu，点击选项-常规设定

2.设置完成效果

3.首先连接手柄，然后点击 Options，Input Settings（使用键盘鼠标请阅读之后进阶教程中的键鼠设置）

下图为设置完成的截图，请按照说明设置

我的为pro手柄，xbox等请按自己abxy键位设置，体感震动等请阅读之后的进阶设置

4. AMD 显卡用户应该点击 Options>Graphic packs>Workarounds>GPU Specific workarounds，把其中 AMD 开头的两项勾选，并取消勾选 Nvidia: Explosion Smoke

5.双击或右击游戏图标，点击 Start，开始加载缓存,一次加载缓存较慢，视电脑配置，时间一般在 5 到 20 分，请耐心等待，第一次加载完成后，再次读取的时间会在 1 分钟以内

  • NVIDIA 显卡用户在缓存加载完成后，应立即检查 Cemu&BotW\Cemu\shaderCache\driver\nvidia\GLCache 文件夹 若文件夹大小低于 100MB（正常约为 120MB 左右）则说明缓存加载不完整，此时进入游戏会频繁的剧烈的卡顿
  • 在较老版本（约 2019/4 之前）的 NVIDIA 显卡驱动中，GLCache 文件 夹的正常大小会在 450MB 左右，这种情况下则可以更新至最新显卡驱 动或者确保 GLCache 文件夹大小不低于 400MB

由于 NVIDIA 驱动已经造成了不同版本缓存大小不一致的情况，上述提供的正常缓存大小仅供参考，缺乏时效性，最终还是以是否存在频繁的剧烈的卡顿作为检验缓存是否加载完整的标准

  • 解决方法
    1. 删除 Cemu\shaderCache\driver\nvidia\GLCache 文件夹
    2. 打开 NVIDIA 控制面板，选择程序设置，点击恢复按钮并点击 应用（如果恢复按钮显示灰色，则点击删除然后再次添加 Cemu.exe）
    3. 重新开始加载缓存，注意删除 GLCache 文件夹之后重新加载 视为第一次加载，时间也是在 5 到 20 分钟之间
    4. 检查 GLCache 文件夹大小，若大小正常则重复初步设置中的 NVIDIA 控制面板设置，设置完成即可正常开始游戏，更新显卡驱动会 导致之前的缓存失效，遇到失效的情况，重复上述步骤即可

常见问题

Q: 帧数不高怎么办

A: 如果软件配置照着教程来的话说明硬件配置不够

Q: 遇到需要体感才能过关的神庙怎么办

A: 请阅读之后的体感设置

Q: 缓存加载完成后崩溃怎么办

A: 确保 Cemu 语言为英文，确保 General settings>MLC Path 为空白

Q: 声音听起来有问题，出现杂音怎么办

A: 调高 Options>General settings>Audio>Latency，直至声音正常为止

Q: 游戏内水面显示为白色怎么办

A: 将图形包 FPS++菜单下的 Fence Method 从右侧选为 Accurate Fence

Q: 手柄设置中点击 Load 之后 Controller API 自动跳转怎么办

A: 跳转后手动选择 XInput 并点击 Save 或者打开 Cemu 根目录的 controllerProfiles 文件夹，将 backup 文件夹中相应的配置文件覆盖到上层

Q: 游戏中无法使用照相机功能，无论怎样对准目标都只出现问号怎么办

A: 暂时开启 General settings>Graphics>Full sync at GX2DrawDone()，注 意开启后可能会一定程度上降低帧数，不用照相机功能时关闭 Full sync at GX2DrawDone() 即可恢复帧数
