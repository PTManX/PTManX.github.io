CEMU体感等进阶设置
2020-02-24|CEMU模拟器
阅读量:|评论数:

我的cemu版本：1.72b，本教程并不完善欢迎大家进群讨论交流补充。本教程涉及到的文件我也会都上传到群文件里，转载请注明出处。

cpu设置

  1. 游戏-右键-edit game profile
  2. 根据自己机器配置选择开启单核双核三核编译 []

体感设置

内容全部来自官方wiki说明，有条件者可自己去官网查看。由于设备限制本人仅能给出平板和pro手柄设置教程，xbox手柄没有体感。ios ps4手柄请自己寻找教程

安卓手机

  1. 点此处下载并在手机上安装motionsource（手机必须有陀螺仪功能）

  2. 确保手机和电脑在同一局域网内，打开app，选择ip地址，其他都默认即可-勾选开启右下方server按钮，下方可根据自己喜好选择手机使用方式。

  3. 在电脑上下载PadTest,打开后运行，左上角输入上一步手机所选ip地址，点击start

  4. 打开cemu安装文件夹-打开cemuhook.ini,输入server ip，

  5. 重新打开Cemu，会发现在选项中的Game Pad motion source选项里dsu1已经有设备连接而不是disconeected，说明设置没问题，勾选后进游戏即可用手机控制体感。

pro手柄和震动设置

设置并不完善，瞄准自动一直往右偏，而且震动还是很弱，触发很少，欢迎懂哥交流一波

  1. 电脑蓝牙连接Pro手柄

  2. 下载解压BetterJoyForCEMU(官方github地址)

  3. 先以管理员身份运行.driver install批处理文件，

    然后再打开运行exe文件，软件会显示出pro手柄，点击lonate，手柄就会震动，说明连接正常。

  4. 打开cemu，在选项中的Game Pad motion source选项里选择pro手柄，体感设置完毕。

  5. 震动设置：打开cemu选项-输入设定-选择配置文件-放大往下拉就会看到额外设定，打开额外设定就能看到震动选项，默认为0，根据自己喜好选择即可，改完记得保存。

帧数设置

cemu选项-图像插件-modes-fps++-static fps mod，即可设置帧数。
