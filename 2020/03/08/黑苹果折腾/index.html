黑苹果折腾记录
2020-03-15|黑苹果
阅读量:|评论数:

前言

​ 在这个史上最长假期里，在家实在闲的蛋疼，就折腾了很多东西。n1电视盒子旁路由，我家云，树莓派软路由。。不过这些教程很多也不难，于是我就决定挑战下一个目标：黑苹果。半年前我已经装过一次，不过那次只是为了尝鲜装完就又删掉了，这次我决定好好折腾一下搞完美点，没想到一折腾就是五六天，写篇文章记录下，希望对大家有点帮助。许多过于基础的问题我不会提，网上一堆教程大家可以自己按需查找。

必备软件

windows

  • Diskgenius：很强大的磁盘管理工具，装不装黑苹果都是必备的。
  • booticex64：管理UEFI启动选项。其他easyuefi啥的也行
  • balenabtcher：制作系统盘工具

macos

  • cloverconfigurator：后续优化
  • hackintool：显示器补丁，声卡补丁，usb定制等。

网站资源

黑果小兵博客上的各种教程，github，远景论坛，b站。

过程

1.准备EFI文件

​ 这一步是最重要的，大家可以在远景和github搜索自己笔记本/台式主板的关键词。只要不是太古老或太冷门基本都能找得到。我的电脑是神舟的一款游戏本，算是比较冷门的。神舟zx7-cp5s2，蓝天模具桌面处理器i5-8400，主板z370，显卡1060。远景一查只有几个帖子是这个。挑一个日期最新的下载。

2.制作系统盘

​ 最新版系统是15.3，但去年4月的efi系统是14.x版本不一定适配，先下载了14.6，下载系统镜像-balenaEtcher写入u盘。

打开diskgenius–选择u盘esp分区，将自己准备的efi文件替换进去，启动u盘就搞好了。

另外有些人的电脑可能太老，需要修改为uefi引导，修改硬盘分区表等，大家可根据自己需要网上查找。

3.系统安装

​ 这一步没什么好说的，各种详细的教程很多。由于我的bios设置太简单甚至简陋，就在网上找了解锁版bios刷了一下，换掉了祖传的开机战神logo。另外第一次进入macos读条时，如果你的EFI文件不对问题很大，就可能出现各种乱七八糟的问题。自己处理难度很高，问别人也很麻烦，百度远景csdn啥的查一下解决不了的话就可以放弃了。我自己安装时出现了禁止标志，换了一个2.0u盘插入2.0口就解决了，成功进入安装页面。照着步骤来成功安装启动。

4.更换为硬盘引导启动（双系统为例）

  • 进入windows系统，打开diskgenius，查看windows安装盘的esp分区。如果小于200m，需要扩大一下，我是把esp分区挨着的mbr分区直接删除，空间分给了esp分区。然后将里面的microsoft文件夹复制出来移到efi文件夹内—esp分区内文件都删除—-将efi文件夹复制进esp分区。

  • 然后打开bootice软件—-分区管理—–给esp分区分配一个任意盘符，这样就能在我的电脑里看到。

  • 还是bootice软件—–选择上方UEFI选项（如果有人不知道怎么进入bios直接勾选uefi页面的重启进入bios页面就可以了）—修改启动序列—-添加—–选择刚才给esp分区的盘符–选择EFI-CLOVER文件夹内的cloverx64—保存—上移到第一位—-保存当前启动项设置。这样就完成了，下一次开机就会进入clover页面，可以选择自己想要启动的系统

到了这一步，如果efi文件比较完美，黑苹果的各种功能都正常，也不想升级系统，那么黑苹果安装就已经完成了，想要折腾的可以继续往下看

5.系统升级15.3问题解决

我抄来的EFI文件除了显卡网卡这两个无解问题外，休眠触摸板屏幕亮度触摸板啥的都挺完美，appstore也能登录，我就直接在appstore里升级了15.3版本。然后就出了很多问题…

No.1 安装黑屏

  • 更新WhateverGreen.kext到v1.3.4
  • 开机clover引导界面选择options（也可按o直接进入），选中第一行boot args在最后添加引导参数：agdpmod=pikera

No.2 声卡注入失灵

这个问题折磨了我两天，开关机无数次，用万能声卡可以驱动，但一旦换成applealc注入就又失灵了。hackintool也识别不出声卡。黑锅小兵的hackintool注入声卡教程也试了没用。最后终于找到了解决方案：

  • 更新applealc和lilu到最新版
  • 禁用config文件acpi补丁里的两个hdef改名补丁，修复勾选HPET IPIC（这个不一定，可以自己试试）。
  • 引导参数添加-lilubetaall
  • 声卡注入里选择对应自己声卡型号的id，生成保存config文件

以上操作都是在clover configurator中进行，我用这个方案后声卡驱动完美，休眠重启也能正常使用。麦克风输入正常,siri也能使用。

No.3 更换无线网卡

笔记本无线网卡不能用实在难受，我就照着黑锅小兵的大神推荐在淘宝买了一块dw1820a，买的时候110第二天就涨到150了，拆机后换下原来的网卡，先进入windows装好提前下载的驱动，无线蓝牙都正常（我的机器不需要屏蔽针脚，有的人可能需要）。然后进入macos。这就出问题了，开机卡爆，wifi不断开启关闭，照着大神的的解决方案安装了一个驱动，添加了引导参数。还是卡爆。网卡有三种注入方法，我选的是第三种hackintool打补丁，大家搞得时候可以在装网卡前提前用config注入的方法把补丁打好。我是已经装上了懒得拆机在拆下来，就直接硬顶着卡顿，开关机十几次争分夺秒，在死机前成功打好补丁，重启后终于正常了，安装驱动后，蓝牙aurdrop啥的也能正常使用。

说一下我的步骤：

打开hackintool—–应用补丁——应用补丁—-勾选下图几项—-生成补丁—左上方菜单栏选择文件—-导出—–config，重启ok

N0.4 修复显卡显存型号等

  • 打开hackintool—–应用补丁——根据自己的cpu架构和显卡型号下方选择合适的产品ID

  • 应用补丁–按下图勾选

  • 高级—–按下图勾选—–生成补丁—–按上一步操作导出到config

  • 重启。一切ok

No.5 USB定制

这个黑锅小兵大神的教程很详细，我照着做完也没出问题。大家可以看大神的教程。

6.总结

​ 笔记本装黑苹果就不能怕折腾，小白真想搞建议买电脑前就挑选小米联想这种折腾的人多的，不然很麻烦。以上就是我这次安装过程遇到的各种问题，另外谁说神船容易翻？这几天折腾黑苹果电脑高强度开关机上百次，我自己都害怕结果一点问题没出，神船🐂皮！我为神船代言！

​ 目前我还有两个小问题没解决，显示器亮度无法调节，这个无伤大雅。还有一个问题是hdmi没输出。我在win下试了试，独显一禁用hdmi就没输出了，所以可能是游戏本的问题，也欢迎懂哥指点一波，

​ 最后希望这篇文章对大家有所帮助，都能吃上黑苹果。
